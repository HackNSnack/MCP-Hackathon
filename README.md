# **ðŸš€ The HoloNet Protocol: A Star Wars AI Hackathon**

Welcome, Rebel developer\! The HoloNet, the galaxy's information network, has been restored... but it needs a brain.

Your mission is to build an AI agent that can intelligently query the Star Wars API (SWAPI). We will use **FastMCP (Model Context Protocol)** to give a Large Language Model (LLM) the "tools" it needs to become a true Jedi Master of data.

## **The Path to Knighthood**

You will complete 5 levels, each granting your AI new abilities:

* Level 1: First Contact  
  Teach your AI to find a person. ("Tell me about Luke Skywalker.")  
* Level 2: Connecting the Dots  
  Teach your AI to follow relationships. ("What is the name of Luke's homeworld?")  
* Level 3: Strength in Numbers  
  Teach your AI to handle lists of items. ("What films was R2-D2 in?")  
* Level 4: Who's Faster?  
  Teach your AI to compare data. ("Which ship is more expensive: an X-Wing or the Millennium Falcon?")  
* Level 5: Mission Planner  
  Combine all tools to answer complex, narrative questions. ("I'm on Hoth and need an escape plan. Find me a pilot, a fast ship, and a safe desert planet to hide on.")

## **Setup & Launch**

1. **Create an environment:**  
   python \-m venv venv  
   source venv/bin/activate  \# (or venv\\Scripts\\activate on Windows)

2. **Install dependencies:**  
   pip install \-r requirements.txt

3. **Run the HoloNet Server:**  
   python server.py

Your server is now live at http://127.0.0.1:8000/mcp.

Connect your LLM client, and...
